##### condition
- 여러 개의 글자 중 하나만 다른 글자(정답 글자)로 채운다.
- 정답 글자 위치는 매번 달라져야 한다.
- 정답 글자를 선택하면 다음 stage로 넘어간다.
- 정답이 아닌 글자를 선택하면 화면의 변화가 없다.
